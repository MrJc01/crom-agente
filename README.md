# crom-agente

O orquestrador central do ecossistema CromIA. Este daemon atua como o motor de Inteligência Artificial de alta performance, comunicando-se com modelos remotos (via OpenAI, Anthropic, Gemini, OpenRouter) e gerenciando ferramentas, arquivos locais, agentes autônomos e execução de scripts na sua máquina.

Para acessar a documentação completa da arquitetura, referências técnicas e manuais de uso detalhados, acesse a pasta **[docs/](docs/)**.
